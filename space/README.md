# Universal π Clock

Interactive **Egg of Life** sacred-geometry clock: seven cascading gears, 350 π-labeled slices per circle, hand indicators, and real-time Earth-rate mode.

- [GitHub](https://github.com/kinaar8340/universal_clock)
- Gear 1 (center) = fastest; outer ring = Gears 2–7 clockwise from ~1 o'clock
- Batch advance or real-time simulation with adjustable speed multiplier
